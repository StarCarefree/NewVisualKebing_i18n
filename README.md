# NewVisualKeybing

## 中文

NewVisualKeybing 是一个面向 Minecraft 的可视化按键绑定管理模组。它通过键盘和鼠标布局视图，帮助玩家快速查看、筛选和调整当前整合包中的按键占用。

### 功能

- 可视化显示键盘、鼠标与滚轮输入状态
- 按模组、状态和搜索关键词筛选按键绑定
- 显示空闲、本模组、其他模组与冲突绑定
- 在详情面板中快速解绑、重绑和调整优先级
- 支持按键方案保存、重命名、应用、导入和导出
- 支持多种键盘布局与隐藏非选中模组按键

### 打开方式

默认按 `K` 打开可视化按键查看器，也可以从控制设置界面进入。

## English

NewVisualKeybing is a visual key binding management mod for Minecraft. It presents keyboard and mouse layouts so players can inspect, filter, and adjust key usage across a modpack.

### Features

- Visual keyboard, mouse, and wheel input status
- Filter bindings by mod, status, and search text
- Show free, current-mod, other-mod, and conflicting bindings
- Quickly unbind, rebind, and edit priority from the details panel
- Save, rename, apply, import, and export key binding profiles
- Multiple keyboard layouts and optional hiding of non-selected mod keys

### Opening

Press `K` by default to open the visual keybind viewer, or access it from the controls screen.
